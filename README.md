# Computer-Vision-Deep-Learning-Stack
Computer Vision and Vision-AI implementations covering classical image processing, deep learning (CNNs), object detection (YOLO, Faster R-CNN), segmentation (UNet, DeepLab, Mask R-CNN), tracking (SORT, DeepSORT), Vision Transformers (ViT), and Generative AI models (CLIP, SAM, Stable Diffusion) using PyTorch ,OpenCV ,NumPy, and modern ML tools.
